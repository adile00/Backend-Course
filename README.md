# Backend-Course

A collection of materials, projects, and notes from a 7-month backend development course.

---

## 📚 Course Overview

This repository documents the journey through backend development, covering core concepts, tools, and technologies used to build scalable, secure, and maintainable server-side applications.

---

## 🗓️ Monthly Breakdown

### Month 1 – Programming Fundamentals & Git
- Variables, data types, control flow, functions
- Object-Oriented Programming (OOP) principles
- Version control with Git & GitHub
- Branching, merging, pull requests

### Month 2 – Web Basics & HTTP
- How the web works (client–server model, DNS, TCP/IP)
- HTTP methods: GET, POST, PUT, PATCH, DELETE
- Status codes and headers
- REST API principles

### Month 3 – Node.js & Express.js
- Introduction to Node.js and the event loop
- Building REST APIs with Express.js
- Middleware, routing, and error handling
- Environment variables and configuration (dotenv)

### Month 4 – Databases
- Relational databases with PostgreSQL / MySQL
  - SQL queries, joins, transactions, indexing
- NoSQL databases with MongoDB
  - Collections, documents, queries, aggregation
- ORM / ODM: Sequelize, Mongoose

### Month 5 – Authentication & Security
- User registration and login flows
- Password hashing with bcrypt
- JWT (JSON Web Tokens) – access & refresh tokens
- OAuth 2.0 basics
- Security best practices (CORS, Helmet, rate limiting, input validation)

### Month 6 – Advanced Backend Topics
- File uploads (Multer, cloud storage)
- Email sending (Nodemailer / SendGrid)
- Background jobs & queues (Bull / BullMQ)
- Caching with Redis
- WebSockets and real-time communication (Socket.io)

### Month 7 – Testing, Deployment & DevOps
- Unit and integration testing (Jest, Supertest)
- API documentation (Swagger / OpenAPI)
- Docker & containerization
- CI/CD pipelines (GitHub Actions)
- Cloud deployment (AWS / Heroku / Railway)
- Monitoring and logging basics

## 🚀 Getting Started

Each monthly folder contains its own README with setup instructions. In general:

```bash
# Clone the repository
git clone https://github.com/adile00/Backend-Course.git
cd Backend-Course

# Navigate to a specific module
cd month-3-nodejs-express

# Install dependencies
npm install

# Start the development server
npm run dev
```
